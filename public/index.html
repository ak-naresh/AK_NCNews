GET /api/topics

Description:

Retrieves a list of all available topics on NC News. Each topic includes a slug and description.

Query Parameters:

None

Example Response:

Status: 200 OK

{
  "topics": [
    {
      "slug": "coding",
      "description": "Code is love, code is life"
    },
    {
      "slug": "football",
      "description": "FOOTIE!"
    },
    {
      "slug": "cooking",
      "description": "Hey good looking, what you got cooking?"
    }
  ]
}

Expected Error Codes:

  • 404 Not Found – Invalid endpoint path
  • 500 Internal Server Error – Unexpected server error

GET /api/articles

Description:

Retrieves a list of all articles published on NC News. Articles are sorted by creation date (newest first) and include a comment count for each article.

Query Parameters:

None (future enhancements may include filtering and sorting options)

Response Properties:

  • author - Username of the article author
  • title - Article title
  • article_id - Unique article identifier
  • topic - Topic category
  • created_at - ISO timestamp of creation
  • votes - Current vote count
  • article_img_url - URL of article image
  • comment_count - Number of comments (integer)

Example Response:

Status: 200 OK

{
  "articles": [
    {
      "author": "jessjelly",
      "title": "Running a Node App",
      "article_id": 1,
      "topic": "coding",
      "created_at": "2020-11-07T06:03:00.000Z",
      "votes": 0,
      "article_img_url": "https://images.pexels.com/photos/11035380/pexels-photo-11035380.jpeg?w=700&h=700",
      "comment_count": 8
    },
    {
      "author": "jessjelly",
      "title": "The Rise Of Thinking Machines",
      "article_id": 2,
      "topic": "coding",
      "created_at": "2020-05-14T04:15:00.000Z",
      "votes": 0,
      "article_img_url": "https://images.pexels.com/photos/373543/pexels-photo-373543.jpeg?w=700&h=700",
      "comment_count": 3
    }
  ]
}

Expected Error Codes:

  • 404 Not Found – Invalid endpoint path
  • 500 Internal Server Error – Unexpected server error

GET /api/articles/:article_id

Parametric Endpoint:

:article_id - Integer representing the article's unique identifier

Description:

Retrieves a single article by its ID. Returns complete article information including title, body, topic, author, creation date, vote count, and image URL.

Example Request:

GET /api/articles/1

Example Response:

Status: 200 OK

{
  "article": {
    "article_id": 1,
    "title": "Running a Node App",
    "body": "This is part two of a series on how to get up and running with Systemd and Node.js. This part dives deeper into how to successfully run your app with systemd long-term, and how to set it up in a production environment.",
    "topic": "coding",
    "author": "jessjelly",
    "created_at": "2020-11-07T06:03:00.000Z",
    "votes": 0,
    "article_img_url": "https://images.pexels.com/photos/11035380/pexels-photo-11035380.jpeg?w=700&h=700"
  }
}

Expected Error Codes:

  • 400 Bad Request – Invalid article_id format (must be a number)
  • 404 Not Found – Article with specified ID does not exist
  • 500 Internal Server Error – Unexpected server error

GET /api/articles/:article_id/comments

Parametric Endpoint:

:article_id - Integer representing the article's unique identifier

Description:

Retrieves all comments for a specific article. Comments are sorted by creation date (newest first). Each comment includes comment ID, body, author, vote count, and creation timestamp.

Example Request:

GET /api/articles/1/comments

Example Response:

Status: 200 OK

{
  "comments": [
    {
      "comment_id": 5,
      "body": "I hate streaming noses",
      "article_id": 1,
      "author": "icellusedkars",
      "votes": 0,
      "created_at": "2020-11-03T21:00:00.000Z"
    },
    {
      "comment_id": 2,
      "body": "The beautiful thing about treasure is that it exists...",
      "article_id": 1,
      "author": "bainesface",
      "votes": 14,
      "created_at": "2020-10-31T03:03:00.000Z"
    }
  ]
}

Expected Error Codes:

  • 400 Bad Request – Invalid article_id format (must be a number)
  • 404 Not Found – Article not found or no comments exist for this article
  • 500 Internal Server Error – Unexpected server error

POST /api/articles/:article_id/comments

Parametric Endpoint:

:article_id - Integer representing the article's unique identifier

Description:

Adds a new comment to a specific article. Requires authentication via username and a comment body. Returns the newly created comment with all its properties.

Request Body Format:

{
  "username": "string (required)",
  "body": "string (required)"
}

Example Request:

POST /api/articles/1/comments

{
  "username": "tickle122",
  "body": "This is a fantastic article! Thanks for sharing."
}

Example Response:

Status: 201 Created

{
  "comment": {
    "comment_id": 19,
    "body": "This is a fantastic article! Thanks for sharing.",
    "article_id": 1,
    "author": "tickle122",
    "votes": 0,
    "created_at": "2026-01-09T12:34:56.789Z"
  }
}

Expected Error Codes:

  • 400 Bad Request – Invalid article_id format or missing required fields (username/body)
  • 404 Not Found – Article not found or username does not exist
  • 500 Internal Server Error – Unexpected server error

DELETE /api/comments/:comment_id

Parametric Endpoint:

:comment_id - Integer representing the comment's unique identifier

Description:

Permanently deletes a comment by its ID. Successfully deleted comments return no content in the response body.

Example Request:

DELETE /api/comments/5

Example Response:

Status: 204 No Content

(No response body)

Expected Error Codes:

  • 400 Bad Request – Invalid comment_id format (must be a number)
  • 404 Not Found – Comment with specified ID does not exist
  • 500 Internal Server Error – Unexpected server error

PATCH /api/articles/:article_id

Parametric Endpoint:

:article_id - Integer representing the article's unique identifier

Description:

Updates the vote count for a specific article. The inc_votes value can be positive (to upvote) or negative (to downvote). Returns the updated article with the new vote count.

Request Body Format:

{
  "inc_votes": integer (required)
}

Example Request:

PATCH /api/articles/1

{
  "inc_votes": 1
}

Example Response:

Status: 200 OK

{
  "article": {
    "article_id": 1,
    "title": "Running a Node App",
    "body": "This is part two of a series...",
    "topic": "coding",
    "author": "jessjelly",
    "created_at": "2020-11-07T06:03:00.000Z",
    "votes": 1,
    "article_img_url": "https://images.pexels.com/photos/11035380/pexels-photo-11035380.jpeg?w=700&h=700"
  }
}

Expected Error Codes:

  • 400 Bad Request – Invalid article_id or inc_votes format (must be numbers)
  • 400 Bad Request – Missing inc_votes field in request body
  • 404 Not Found – Article with specified ID does not exist
  • 500 Internal Server Error – Unexpected server error
