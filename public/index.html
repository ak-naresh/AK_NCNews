GET /api/topics

Description:

Responds with a list of all available topics on NC News. Each topic includes a slug, description, and image URL.

Example Response:

{ topics: [
  { description: "Code is love, code is life",
    slug: "coding",
    img_url: "",
  },
  { description: "FOOTIE!",
    slug: "football",
    img_url: "https://images.pexels.com/photos/209841/pexels-photo-209841.jpeg?w=700&h=700",
  }
]}

Expected Error Codes:

  • 400 Bad Request – Invalid quert
  • 404 Not Found – Resource not found
  • 500 Internal Server Error – Unexpected server error

  {title: "The Rise Of Thinking Machines: How IBM's Watson Takes On The World",
    topic: "coding",

    body: "Many people know Watson as the IBM-developed cognitive super computer that won the Jeopardy! gameshow in 2011. In truth, Watson is...",
      created_at: new Date(1589418120000),
    votes: 0,
    article_img_url: "https://images.pexels.com/photos/373543/pexels-photo-373543.jpeg?w=700&h=700",

Expected Error Codes:

  • 400 Bad Request – Invalid query

GET /api/articles/:article_id

Parametric Endpoint:

:article_id

Description:

Responds with an article by article ID Response includes article's: title, body, topic, author, creation date, vote count, and image URL.

Example Response:

{ article: [
    article_id: 1,
    title: "Running a Node App",
    body: "This is the content...",
    topic: "coding",
    author: "jessjelly",
    created_at: new Date(1604728980000),
    votes: 0,
    article_img_url: "https://images.pexels.com/photos/11035380/pexels-photo-11035380.jpeg?w=700&h=700"
]}

Expected Error Codes:

  • 400 Bad Request – Invalid article ID (e.g., wrong data type, 22P02 PSQL error for invalid syntax)
  • 404 Not Found – Article not found
  • 500 Internal Server Error – Unexpected server error

GET /api/articles/:article_id/comments

Parametric Endpoint:

:article_id

Description:

Responds with a list of comments by article ID. Each comment includes: comment ID, body, author, vote count, and creation date.

Example Response:

{ comments: [
  { comment_id: 1,
    body: "Great article!",
    article_id: 1,
    author: "tickle122",
    votes: 16,
    created_at: new Date(1604728980000)
  },
  { comment_id: 2,
    body: "Very informative.",
    article_id: 1,
    author: "grumpy19",
    votes: 3,
    created_at: new Date(1604729000000)
  }
]}

Expected Error Codes:

  • 400 Bad Request – Invalid article ID (e.g., wrong data type, 22P02 PSQL error for invalid syntax)
  • 404 Not Found – Article not found or no comments for article
  • 500 Internal Server Error – Unexpected server error

POST /api/articles/:article_id/comments

Parametric Endpoint:

:article_id

Description:

Add a comment to an article by article ID. Requires both the username of the commenter and the comment body. The response returns the created comment with its ID, author, vote count, and date.

Expected format for Requested Body:

  { 
  username:  string,
  body:  string
  }

Example Response:

{ comment: [
    comment_id: 3,
    body: "Thanks for the article!",
    article_id: 1,
    author: "happyreader",
    votes: 0,
    created_at: new Date(1604729100000)
]}

Expected Error Codes:

  • 400 Bad Request – Invalid article ID (e.g., wrong data type, 22P02 PSQL error for invalid syntax)
  • 400 Bad Request – Missing field (username and/or body missing)
  • 404 Not Found – Article not found
  • 404 Not Found – Username not found
  • 500 Internal Server Error – Unexpected server error

DELETE /api/comments/:comment_id

Parametric Endpoint:

:comment_id

Description:

Deletes a comment by comment ID. Responds with status 204 and no content.

Expected Error Codes:

  • 400 Bad Request – Invalid comment ID (e.g., wrong data type, 22P02 PSQL error for invalid syntax)
  • 404 Not Found – Comment not found
  • 500 Internal Server Error – Unexpected server error

PATCH /api/articles/:article_id

Parametric Endpoint:

:article_id

Description:

Updates vote count for an article ID. Requires a request body with an inc_votes integer to increment or decrement current vote total. Response returns updated article with new vote count.

Expected format for Requested Body:

  { 
  inc_votes:  integer
  }

Example Response:

{ article: [
    article_id: 1,
    votes: 1
]}

Expected Error Codes:

  • 400 Bad Request – Invalid article ID (e.g., wrong data type, 22P02 PSQL error for invalid syntax)
  • 400 Bad Request – Missing field in body (inc_votes missing)
  • 404 Not Found – Article not found
  • 500 Internal Server Error – Unexpected server error
